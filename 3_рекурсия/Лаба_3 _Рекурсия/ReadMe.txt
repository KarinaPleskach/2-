Задачу, выданную преподавателем, необходимо  решить двумя способами - с помощью рекурсии и без нее.
Результат работы программы -  ответ, полученный с помощью рекурсии, а также ответ, полученный
 с помощью итерационного решения.

 
Для успешной защиты  работы необходимо:
- изучить теорию по теме работы, а также примеры рекурсивных подпрограмм;

- написать программу, опираясь не требования, перечисленные выше,и выполнить       тестирование;

- на листе формата А4 изобразить графически рекурсивные вызовы на примере своей   задачи,
 используя соответсвующее компьютрное ПО (пример графического изображения в файле
 "сущность рекурсии"). Перечислить все свойства   рекурсии для своей задачи.
 Распечатать и взять на защиту; 

- после успешной демонстрации программы преподавателю, ответить на вопросы, 
  касающиеся кода (алгоритм, структуры данных и т.д.);

- выполнить письменное задание по теме работы БЕЗ ОШИБОК.
